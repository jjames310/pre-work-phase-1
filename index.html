Pre-work Phase 1

Git is version control software on your computer.

We COMMIT our code versions to git on our computer.

Github.com is a cloud service that works with git to help us collaborate and share our code with others.

We PUSH our code to github.com.

We DEPLOY our projects to a server that the world can view them in a browser on a computer, tablet, or phone.

In the bootcamp we DEPLOY to the bootcamp-coders.cnm.edu server

DEPLOY and PUSH are DIFFERENT!!!
